# Supermercado RB

## Objetivo
Creación de una tienda online para la venta de productos de primera necesidad para la empresa de Supermercados RB

## Herramientas usadas
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Git y GitHub
- Visual Studio Code

## Funcionalidades
- Página principal
- Catálogo de productos
- Categorías de productos
- Menú de navegación
- Carrito de compras visual
- Botones para agregar productos
- Diseño responsive

## Pruebas realizadas
- Revisé que las páginas abran correctamente.
- Revisé que los enlaces funcionen.
- Revisé que las imágenes carguen.
- Revisé que los botones de agregar respondan.
- Revisé el diseño en tamaño de celular y computadora.

## Limitaciones
- No tiene pago real.
- No tiene base de datos.
- No tiene inicio de sesión.
- El carrito es una simulación básica.
